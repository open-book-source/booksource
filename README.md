## 开发

### 连接开发设备
```
⌘（command）+ ⌃（control）+ F6
```

### 测试
打开需要测试的脚本, 例如: `example.com.js`, 然后按`F6`

### 生成 `repository.json` 文件

```shell
npm run generate
```
